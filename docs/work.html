STATIC SITE GENERATOR Personal

2018 - PRESENT

When I needed to start this website/blog I figured it would be a fun opportunity to build my own static site generator to power it. I write my posts in markdown, and run a build script which uses Showdownjs to convert that markdown to HTML. The final output is just a collection of HTML, CSS, and JS files, which can then be hosted very simply. This site is hosted for free on GitHub Pages

IN its current state the generator and the site content are intrinsically linked within the same repo, but maybe in the future I'll look into separating them out, so that the generator will work with other sites too. The code is pretty content-agnostic already, so it probably wouldn't be much work to do that.

CIRCULAR WAVE LTD Professional

2016 - 2020

I spent almost 4 years working with this healthcare tech startup, both in a technical and non-technical capacity.

I initially spent some time updating and maintaining a pre-existing MVP app built with Ionic/Angular JS, with a Node.js Monolith backend. Later we completely rebuilt the same product using React and React Native, while also splitting our Node.js monolith back end into several microservices.

Working with Circular Wave taught me a huge amount about a lot of things. I was involved in almost every aspect of the business in one way or another, including writing code, planning features and implementations, designing user interfaces, illustration, team leadership, becoming a Professional Scrum Master I (PSM I), overseeing a transition to using Agile methodologies, interviewing, hiring, and firing.

Working closely with a small startup, I rode the highs and lows of the business alongside its founders. I'm convinced that my 4 years of experience working with Circular Wave were worth several more years of experience at a more established company.
